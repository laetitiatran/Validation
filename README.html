la boucle if vérifie si il n'y a pas de Vagrantfile déjà existant. Si oui elle renvoie un message sinon elle en crée un. ensuite on installe les mises à jour ainsi que la virtualbox. Lorsque l'utilisateur n'écrit pas les bons chiffres alors on lui renvoie un message d'erreur.
